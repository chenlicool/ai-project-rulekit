# ai-project-rulekit

通用 AI 项目开发规范启动包（0-1 规则模板）。

## 中文简介
本仓库提供一套可直接复制的规则模板，用于把“影响面判断 → 规则 → 文档 → 代码”的流程固化。适用于从 0 开始的新项目，也适用于已有项目的规范补齐。

## 包含内容
- 规则总纲与分域法典（前端/后端/测试/文档）
- AI 灵魂定义规则（SOUL）
- 统一文档模板（ARCHITECTURE / API / CHANGELOG / MEMORY / README / STACK）
- 0 → 1 初始化与任务完成后的同步工作流
- 中英文版本模板：`template/`（中文）与 `template-en/`（英文）

## 使用方式
1. 复制 `template/` 或 `template-en/` 到你的项目根目录。
2. 填写 `STACK.md`，明确计划技术栈。
3. 执行 `WORKFLOWS/INIT.md`。
4. 按 `AGENTS.md` + `.rules/` 开发与更新文档。

## 重要说明
- `template/README.md` 与 `template-en/README.md` 是项目模板文件，不是本仓库介绍。
- `MEMORY.md` 采用时间倒序，最新记录置顶。
