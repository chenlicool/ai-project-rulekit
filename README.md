# ai-project-rulekit

通用 AI 项目开发规范启动包（0-1 规则模板）。

A universal starter kit for AI project rules and documentation from day 0.

**中文简介**
本仓库提供一套可直接复制的规则模板，用于把“影响面判断 → 规则 → 文档 → 代码”的流程固化。适用于从 0 开始的新项目，也适用于已有项目的规范补齐。

**English Summary**
This repository provides a copy‑ready rulekit to enforce an Impact → Rules → Docs → Code workflow. It fits brand‑new projects and can retrofit existing ones.

**包含内容**
- 规则总纲与分域法典（前端/后端/测试/文档）
- 统一文档模板（ARCHITECTURE / API / CHANGELOG / MEMORY / README / STACK）
- 0 → 1 初始化与任务完成后的同步工作流

**What's Inside**
- Global rules and domain rulebooks (frontend/backend/test/docs)
- Document templates (ARCHITECTURE / API / CHANGELOG / MEMORY / README / STACK)
- Initialization and post‑task sync workflows

**使用方式**
1. 复制 `template/` 到你的项目根目录。
2. 填写 `STACK.md`，明确计划技术栈。
3. 执行 `WORKFLOWS/INIT.md`。
4. 按 `AGENTS.md` + `.rules/` 开发与更新文档。

**Quick Start**
1. Copy `template/` into your project root.
2. Fill `STACK.md` with your planned stack.
3. Follow `WORKFLOWS/INIT.md`.
4. Use `AGENTS.md` and `.rules/` for all changes.

**重要说明**
- `template/README.md` 是项目模板，请勿当作本仓库介绍。
- `MEMORY.md` 采用时间倒序，最新记录置顶。

**Notes**
- `template/README.md` is a template for target projects, not this repo.
- `MEMORY.md` is maintained in reverse chronological order.
